# 人可阜東有限公司選股邏輯整理

## 1. 核心觀念

本選股模型不是猜題材，而是用公開資料快速找出兩種股票：

1. 公司真的在成長：看單月營收年增率 YoY、營收是否創高。
2. 市場真的有錢進來：看成交金額、法人買賣超、股價動能。

核心公式：

```text
好標的 = 營收成長確認 + 市場金流確認
```

最重要的基本面指標是：

```text
單月營收 YoY
```

因為營收成長通常有延續性。公司若產品供不應求、訂單增加、產業趨勢成立，營收不太可能只好一個月就結束。

## 3. 營收排序優先級

營收邏輯的重點是：

```text
營收創高優先於單純高 YoY
```

排序優先級：

```text
1. 營收創歷史新高，且 YoY >= 100%
2. 營收創歷史新高，且 YoY >= 30%
3. 單月營收 YoY >= 200%
4. 單月營收 YoY >= 100%
5. 單月營收 YoY >= 30%
6. 單月營收 YoY 0% ~ 30%
7. 單月營收 YoY < 0%
```

範例判斷：

```text
創高 + YoY 92.7% > YoY 171.6% 但未創高
```

也就是：

```text
創高 + 30% 以上，優先級可以高於單純 YoY 200%
```

## 4. 三種選股模式

### 模式 1：嚴格模式

目的：

```text
選出營收成長明確，且市場資金共識最高的股票。
```

適合：

```text
波段操作
較大部位
想降低亂追題材風險
```

硬條件：

```text
單月營收 YoY > 30%
成交金額不可太低
排除處置股
```

排序：

```text
1. 是否營收創歷史新高
2. 單月營收 YoY
3. 成交金額
4. 三大法人合計買賣超
5. 股價動能
```

優點：

```text
容易選出基本面成長 + 金流強勁的主流股。
```

缺點：

```text
條件嚴格，有時候可能沒有適合標的。
```

### 模式 2：成長模式

目的：

```text
選出當下營收成長最強的好公司。
```

適合：

```text
中長波段
成長股策略
不想太在意短線成交金額
```

硬條件：

```text
單月營收 YoY > 30%
排除明顯風險股
```

排序：

```text
1. 是否營收創歷史新高
2. 單月營收 YoY
3. 連續 YoY > 30% 的月份數
4. 近 3 個月營收趨勢
5. 成交金額
```

優點：

```text
容易找到持續大漲小回的成長股。
```

缺點：

```text
如果短期沒有資金關注，不一定馬上漲。
```

### 模式 3：金流模式

目的：

```text
選出市場當下最有資金共識的股票。
```

適合：

```text
短線
強勢日策略
日拉回策略
強勢週策略
```

硬條件：

```text
成交金額排名前段
流動性足夠
```

排序：

```text
1. 成交金額
2. 成交金額相對 20 日均值放大倍數
3. 漲跌幅
4. 外資 / 投信 / 三大法人買賣超
5. 股價動能
```

優點：

```text
容易找到市場正在交易的主流標的。
```

缺點：

```text
可能選到營收不強、純題材或炒作型股票。
```

## 5. 阜東分數

建議用 0 到 100 分。

```text
阜東分數 = 營收分 + 金流分 + 籌碼分 - 風險扣分
```

權重：

```text
營收分：50 分
金流分：30 分
籌碼分：20 分
風險扣分：最多 -30 分
```

### 營收分

```text
營收創歷史新高 + YoY >= 100%：50 分
營收創歷史新高 + YoY >= 30%：45 分
YoY >= 200%：40 分
YoY >= 100%：35 分
YoY >= 30%：30 分
YoY 0% ~ 30%：10 分
YoY < 0%：0 分
```

### 金流分

用成交金額排名計算：

```text
成交金額前 10%：30 分
成交金額前 20%：24 分
成交金額前 30%：18 分
成交金額前 50%：10 分
其他：0 分
```

若有 20 日均成交金額，可以加上放大倍數：

```text
成交金額 > 20 日均值 2 倍：+5 分
成交金額 > 20 日均值 1.5 倍：+3 分
成交金額 < 20 日均值：-3 分
```

### 籌碼分

```text
外資買超：+5 分
投信買超：+7 分
自營商買超：+3 分
三大法人合計買超：+5 分
```

投信權重略高，因為投信偏向波段持股，對中短波段更有參考價值。

### 風險扣分

```text
處置股：-20 分
近 20 日漲幅 > 40%：-10 分
近 20 日漲幅 > 60%：-20 分
YoY < 0 且成交金額暴增：標記「題材 / 炒作風險」
股價跌破 20 日線：-5 分
```

## 6. 標籤系統

每檔股票輸出一個主標籤。

```text
A 級成長金流：營收創高 + YoY > 30% + 成交金額強 + 法人買超
B 級成長：YoY > 30%，但成交金額普通
C 級金流：成交金額強，但營收不強
題材風險：營收 YoY 不佳，但成交金額暴增
觀望：營收普通、成交普通
避開：YoY 衰退、處置、漲幅過熱
```

## 7. 模式輸出邏輯

### 嚴格模式輸出

只顯示：

```text
YoY > 30%
成交金額排名前 30%
非處置股
```

排序：

```text
阜東分數由高到低
```

### 成長模式輸出

只顯示：

```text
YoY > 30%
或營收創歷史新高
```

排序：

```text
營收創高優先
再看 YoY
再看連續成長月份
```

### 金流模式輸出

只顯示：

```text
成交金額排名前 20% 或前 30%
```

排序：

```text
成交金額由高到低
再看成交金額放大倍數
再看法人買賣超
```

## 8. 個股入選理由

每檔股票應該顯示 2 到 4 條理由。

範例：

```text
營收創歷史新高，YoY +92.7%
成交金額 144.1 億，市場資金集中
投信買超 5,200 張，法人籌碼偏多
近 20 日漲幅 28%，尚未過熱
```

若是風險股：

```text
成交金額暴增，但營收 YoY -12.4%
可能屬於題材或短線炒作
近 20 日漲幅 63%，追價風險高
```

## 9. 實作建議

第一階段：

```text
新增三種模式切換
新增阜東分數
新增 A/B/C/風險標籤
把目前成交金額、法人買賣超納入分數
```

第二階段：

```text
接月營收資料
計算單月營收 YoY
計算是否營收創歷史新高
加入營收排序優先級
```

第三階段：

```text
加入 20 日均線
加入 20 日漲幅
加入成交金額放大倍數
加入處置股排除
```

第四階段：

```text
個股頁加入入選理由
加入風險提示
加入模式說明
```

## 10. 預設模式建議

網頁打開時預設使用：

```text
嚴格模式
```

原因：

```text
這是最符合人可阜東選股邏輯的主模式：
營收真的成長，市場也真的有錢進來。
```

使用者可以再切換：

```text
成長模式：偏波段、偏基本面
金流模式：偏短線、偏市場主流
```
